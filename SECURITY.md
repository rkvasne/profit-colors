## Versões Suportadas

Como este é um projeto web (frontend puro), a "versão" é contínua. No entanto, recomendamos sempre utilizar a versão mais recente disponível na branch `main` ou através da página oficial.

| Canal | Suportado |
| --- | --- |
| GitHub Pages (online) | :white_check_mark: |
| Arquivos do repositório (offline) | :white_check_mark: |

## Reportando uma Vulnerabilidade

Levamos a segurança a sério. Se você descobrir uma vulnerabilidade de segurança, por favor **NÃO** abra uma Issue pública.

Em vez disso, envie um e-mail para **rkvasne@gmail.com**.

### O que incluir no reporte:
1. Descrição da vulnerabilidade.
2. Passos para reproduzir.
3. Impacto potencial.

Nós nos comprometemos a:
- Reconhecer o recebimento do seu reporte em até 48 horas.
- Enviar uma estimativa de prazo para correção.
- Notificá-lo quando a correção for lançada.

Obrigado por ajudar a manter o ProfitColors seguro para todos os traders!
